# Todo App (Jetpack Compose)

A simple Todo application built with *Jetpack Compose* using *MVVM architecture* and *StateFlow* for reactive UI updates.

## Features
- Splash screen with animation
- View list of todos
- Add and edit tasks
- Mark tasks as completed
- Clean Material 3 UI
